Adds coloured water. This Fork of Rainbow Source by Shara is compatible with Luanti versions after 5.4.0. I also added buckets for each water color. I'm bad at making textures, so they're generated the same way as bucket_compat from fluid_lib by IcyDiamond. I borrowed bucket_mask.png for that. Bucket crafting recipes are a Setting that is disabled by default. All content contained in this mod is under the MIT License.
